# 🏖️ عقارات رأس البر - Ras El Bar Real Estate Platform

> **منصة عقارية متقدمة لمدينة رأس البر السياحية | Advanced Real Estate Platform for Ras El Bar**

---

## 🌟 نظرة عامة | Overview

**عقارات رأس البر** هي منصة عقارية حديثة وشاملة مصممة خصيصاً لخدمة سوق العقارات في مدينة رأس البر الساحلية. تجمع المنصة بين تكنولوجيا متقدمة وتجربة مستخدم استثنائية، مما يسهل على المشترين والمستأجرين والمستثمرين البحث عن العقارات المثالية.

**Ras El Bar Real Estate** is a modern and comprehensive real estate platform designed specifically for the real estate market in the coastal city of Ras El Bar. The platform combines advanced technology with an exceptional user experience, making it easy for buyers, renters, and investors to find their ideal properties.

## ✨ المميزات الرئيسية | Key Features

### 🔍 نظام البحث المتقدم | Advanced Search System
- **بحث ذكي متعدد المعايير** - ابحث حسب السعر، المساحة، نوع العقار، ونوع العملية (بيع/إيجار)
- **تصفية ديناميكية** - نتائج فورية مع ترتيب متقدم (الأحدث، السعر الأقل، السعر الأعلى)
- **خريطة تفاعلية** - عرض العقارات على خريطة جوجل حية مع معلومات الموقع الدقيقة

### 📱 واجهة مستخدم احترافية | Professional User Interface
- **تصميم متجاوب بالكامل** - تجربة سلسة على الهاتف والتابلت والحاسوب
- **دعم كامل للعربية** - واجهة عربية أصلية مع اتجاه RTL صحيح
- **هوية بصرية فاخرة** - تصميم حديث يعكس طابع المدينة السياحي

### 🏠 إدارة العقارات الشاملة | Comprehensive Property Management
- **إضافة عقارات متقدمة** - نموذج تفصيلي مع رفع صور متعددة وروابط مباشرة
- **تعديل وحذف سهل** - إدارة كاملة للإعلانات مع تحكم فوري
- **صفحة تفاصيل متقدمة** - معرض صور تفاعلي مع Lightbox وخريطة جوجل

### 👥 نظام المستخدمين المتطور | Advanced User System
- **مصادقة آمنة** - OAuth مع Manus للتسجيل الآمن والسهل
- **لوحة تحكم شخصية** - إدارة كاملة للإعلانات والإحصائيات
- **ملف شخصي مرن** - تعديل البيانات الشخصية والتواصل المباشر

### 📊 لوحة إدارة قوية | Powerful Admin Dashboard
- **إحصائيات شاملة** - عدد المستخدمين، الإعلانات، المشاهدات، والاستخدام
- **رسوم بيانية متقدمة** - توزيع العقارات والأسعار والعمليات
- **إدارة المستخدمين** - عرض وتعديل وحذف وترقية المستخدمين
- **إدارة الإعلانات** - تفعيل وإيقاف وحذف الإعلانات بسهولة

### 💬 نظام التواصل المباشر | Direct Communication System
- **زر الاتصال الفوري** - اتصل مباشرة برقم هاتف المعلن
- **تكامل واتساب** - تواصل آمن عبر WhatsApp مع المعلن
- **معلومات المعلن** - عرض كامل بيانات المعلن والتقييمات

## 🛠️ المكدس التقني | Tech Stack

### Frontend
- **React 19** - مكتبة واجهات المستخدم الحديثة
- **TypeScript** - لغة برمجة آمنة وموثوقة
- **Tailwind CSS 4** - نظام تصميم قوي ومرن
- **Wouter** - توجيه خفيف الوزن وسريع
- **shadcn/ui** - مكونات واجهة مستخدم عالية الجودة

### Backend
- **Express 4** - خادم ويب قوي وموثوق
- **tRPC 11** - نظام RPC آمن مع نقل النوع الكامل
- **Node.js** - بيئة تشغيل JavaScript جانب الخادم

### Database
- **MySQL/TiDB** - قاعدة بيانات علائقية قوية
- **Drizzle ORM** - مكتبة ORM حديثة وآمنة من حيث النوع
- **Migrations** - إدارة الإصدارات الآمنة للمخطط

### Authentication & APIs
- **Manus OAuth** - مصادقة آمنة وسهلة
- **Google Maps API** - خرائط تفاعلية وجيوكودينج
- **SuperJSON** - تسلسل آمن للبيانات المعقدة

### Testing & Quality
- **Vitest** - إطار اختبار سريع وحديث
- **TypeScript ESLint** - فحص الكود الثابت

## 🚀 البدء السريع | Quick Start

### المتطلبات | Requirements
- **Node.js** 18+ أو أحدث
- **pnpm** أو npm
- **MySQL/TiDB** قاعدة بيانات

### التثبيت | Installation

```bash
# استنساخ المستودع
git clone https://github.com/yourusername/Ras-elbar-estate.git
cd Ras-elbar-estate

# تثبيت المكتبات
pnpm install

# إعداد متغيرات البيئة
cp .env.example .env.local

# تشغيل الترحيلات
pnpm drizzle-kit generate
pnpm db:migrate

# بدء خادم التطوير
pnpm dev
```

### الوصول إلى التطبيق | Access the Application
- **Frontend**: http://localhost:5173
- **Backend**: http://localhost:3000

## 📚 الوثائق | Documentation

### إضافة عقار جديد | Adding a New Property
1. انتقل إلى صفحة "إضافة إعلان"
2. ملء جميع الحقول المطلوبة (النوع، السعر، المساحة، الوصف)
3. اختر نوع العملية (بيع أو إيجار)
4. أضف صور من جهازك أو روابط مباشرة
5. انقر "نشر الإعلان"

### البحث عن العقارات | Searching for Properties
1. استخدم شريط البحث السريع في الصفحة الرئيسية
2. أو انتقل إلى صفحة "استعرض العقارات"
3. استخدم الفلاتر المتقدمة للبحث الدقيق
4. انقر على أي عقار لعرض التفاصيل الكاملة

### إدارة الإعلانات | Managing Listings
1. انتقل إلى لوحة التحكم الشخصية
2. عرض جميع إعلاناتك مع الإحصائيات
3. عدل أو احذف الإعلانات حسب الحاجة
4. تفعيل أو إيقاف الإعلانات بسهولة
