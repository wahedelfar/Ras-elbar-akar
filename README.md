## 🌟 نبذة عن المشروع

**عقارات رأس البر** هي منصة عقارية رقمية فاخرة وموثوقة تقدم تجربة استثنائية للبحث والشراء والبيع والإيجار للعقارات في مدينة رأس البر الساحلية. نحن نجمع بين التكنولوجيا المتقدمة والخدمة الإنسانية لتسهيل رحلة العقارات الخاصة بك.

### 🎯 رسالتنا
تسهيل عملية البحث والشراء والبيع والإيجار للعقارات من خلال منصة رقمية آمنة وموثوقة، وتمكين المستخدمين باتخاذ قرارات عقارية مستنيرة.

### 🚀 رؤيتنا
أن نصبح المنصة العقارية الرائدة في رأس البر والمنطقة، معروفة بالموثوقية والابتكار وخدمة العملاء المتميزة.

## 🛠️ المكدس التكنولوجي

### Frontend
- **React 19** - مكتبة واجهات المستخدم الحديثة
- **TypeScript** - لغة برمجة آمنة الأنواع
- **Tailwind CSS 4** - نظام تصميم قابل للتخصيص
- **shadcn/ui** - مكونات واجهة مستخدم احترافية
- **Wouter** - توجيه خفيف الوزن
- **tRPC** - اتصال آمن بين Frontend و Backend

### Backend
- **Node.js + Express** - خادم ويب قوي وموثوق
- **tRPC** - إطار عمل RPC آمن الأنواع
- **Drizzle ORM** - مكتبة قاعدة بيانات حديثة

### Database
- **MySQL/TiDB** - قاعدة بيانات موثوقة وسريعة
- **Drizzle Migrations** - إدارة تطور قاعدة البيانات

### Storage
- **AWS S3** - تخزين آمن وموثوق للصور والملفات

### Authentication
- **Manus OAuth** - نظام مصادقة آمن وموثوق

### Analytics & SEO
- **Google Analytics 4** - تتبع شامل للزوار
- **Google Search Console** - أرشفة سريعة في محركات البحث
- **JSON-LD Schema** - بيانات منظمة للمحركات

### Hosting
- **Manus Platform** - استضافة موثوقة وآمنة
- **HTTPS** - اتصال آمن تلقائي

## 📁 هيكل المشروع

```
ras-elbar-realestate/
├── client/                 # تطبيق React
│   ├── src/
│   │   ├── pages/         # صفحات التطبيق
│   │   ├── components/    # مكونات قابلة لإعادة الاستخدام
│   │   ├── contexts/      # React Contexts
│   │   ├── hooks/         # Custom Hooks
│   │   ├── lib/           # مكتبات مساعدة
│   │   └── _core/         # ملفات النواة
│   ├── public/            # ملفات ثابتة
│   └── index.html         # ملف HTML الرئيسي
├── server/                # تطبيق Express + tRPC
│   ├── routers.ts         # تعريفات tRPC
│   ├── db.ts              # مساعدات قاعدة البيانات
│   ├── _core/             # ملفات النواة
│   └── auth.logout.test.ts # اختبارات
├── drizzle/               # إدارة قاعدة البيانات
│   ├── schema.ts          # تعريف الجداول
│   └── migrations/        # ملفات الترحيل
├── storage/               # مساعدات S3
└── shared/                # ملفات مشتركة
```

## 🚀 البدء السريع

### المتطلبات
- **Node.js** 18+
- **npm** أو **pnpm**
- **Git**
- متصفح حديث

### التثبيت

```bash
# استنساخ المستودع
git clone https://github.com/wahedelfar/Ras-elbar-akar.git
cd Ras-elbar-akar

# تثبيت المتطلبات
pnpm install

# إعداد متغيرات البيئة
cp .env.example .env.local

# تشغيل خادم التطوير
pnpm dev
```

### الوصول
- **Frontend:** http://localhost:5173
- **Backend:** http://localhost:3000
